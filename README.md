# ArkChurch

Official website for **ArkChurch** — a global digital church and chapel network. ArkChurch is an initiative of [AppChurch Global Foundation](https://arkchurch.church).

**Live site:** [arkchurch.church](https://arkchurch.church)

## Repository

Static site hosted on GitHub Pages. Key pages:

- `index.html` — Main site
- `editor.html` — Admin editor (`/editor.html`)
- `wad2026.html` — World Alzheimer's Day screening program
- `MCIcommunityguid.html` — MCI community guide
- `MCIpartner.html` — Partnership deck

See `INSTALL.md` for setup and maintenance.
